# Referenzen

Diese Seite sammelt die aktuelle öffentliche Referenzbasis für den Blueprint.
Ziel ist nicht, dass bereits jede Claim-Stufe vollständig bewiesen wäre, sondern Methodik, Governance, Evaluation und Limitations-Sprache an nachvollziehbare Quellen zu binden.

## Zitationsregel

- Inline-Zitate nutzen nummerierte Referenzen wie `[1]`, `[2]`.
- Öffentliche Claims sollen nach Möglichkeit institutionelle oder peer-reviewte Quellen bevorzugen.
- Interne Beobachtungen in diesem Repo bleiben von externen Zitaten getrennt und dürfen nicht als austauschbare Evidenz dargestellt werden.
